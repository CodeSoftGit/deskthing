# DeskThing-pi
Use DeskThing on your Raspberry Pi!

> [!NOTE]
> Not to be confused with [DeskThing-pi by TIMBLOCKER](https://github.com/TIMBLOCKER/DeskThing-Pi)

## What is this?
This is a custom Python and Qt6 based application for DeskThing.

## What do you need?
- Raspberry Pi with Ubuntu Server installed
- Touchscreen display (640x480 or higher)

## Supported models
- Raspberry Pi 5 (5, 500/500+, and Compute Module 5)
- Raspberry Pi 4 (4 Model B, 400, and Compute Module 4/4S)
- Raspberry Pi Zero 2 W [⚠️ Might struggle with some DeskThing apps]

## Installation
1. Clone the repo and cd inside it (Do this on the Raspberry Pi you want to install it on!)
2. Run `sudo bash ./scripts/install.sh`
3. Follow the prompts

```bash
git clone https://github.com/CodeSoftGit/deskthing.git
cd deskthing
sudo bash ./scripts/install.sh
```
